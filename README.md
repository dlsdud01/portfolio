[![GitHub](https://img.shields.io/badge/GitHub-Hamly-181717?style=flat-square&logo=github)](https://github.com/dlsdud01/2025_Graduate_Project)
![Android](https://img.shields.io/badge/Android_Studio-3DDC84?style=flat-square&logo=androidstudio&logoColor=white)

![MQTT](https://img.shields.io/badge/MQTT-660066?style=flat-square&logo=mqtt&logoColor=white)

**소개**
- 라즈베리파이 + MQTT 통신 기반 원격 펫 케어 Android 앱
- 온습도 표시, 조명 제어, 현관문 제어, 카메라 스트림 화면 구성

**담당 역할**
- Activity · Fragment · XML 기반 앱 화면 구성
- MQTT(Paho) 수신 데이터를 UI에 실시간 반영
- `ipcam-view` / `mjpeg-view` / `WebView` 비교 검토 후 MJPEG 스트림 수신 방식 적용

- **하드웨어 제약** → 실제 장비가 완전히 갖춰지지 않은 상황에서 테스트 환경 기반으로 앱-통신-화면 전체 흐름을 구현하고 구조를 검증
